# Bakery Data Visualisation Dashboard

This project analyses European bakery performance using Python.

## Features
- Data cleaning and preprocessing
- Visualisations (bar, pie, scatter, histogram)
- Interactive dashboard using Plotly

## Files
- Dataset (.xlsx)
- Jupyter Notebook (.ipynb)
- Code PDF
- Dashboard PDF

## Tools Used
- Python (Pandas, Matplotlib, Seaborn, Plotly)
